# 매일 매점
> 매점을 편리하게 이용, 관리할 수 있는 웹서비스

"매일 매점에 가는 학생들을 위한 서비스" <br>

## 개발 환경
언어 : Spring, Java, html/css, JS, Bootstrap, Oracle<br>
툴 : STS<br>
DB : Oracle

## 기능
1. 상품 리스트
2. 상품 검색
3. 상품 예약 및 구매
4. 상품 좋아요, 싫어요
5. 상품 추가 및 관리
